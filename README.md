# 戦術ボード — 日本 vs ブラジル (W杯 2026 R32)

スタメン＆ベンチを自分で予想して並べ、フォーメーションと選手の動きを描く Web 戦術ボード。

- 単一の自己完結 HTML（ビルド不要・オフライン動作）
- バニラ JS + SVG、`localStorage` 以外の外部依存なし
- ドラッグ＆ドロップで選手移動／ベンチ交代（常に11人維持）
- フォーメーション10種・チーム独立、矢印/ペン/直線の描画、Undo
- JSON 入出力・PNG 出力・自動保存

## デプロイ

Vercel はルートの `index.html` を静的配信します。ビルド設定は不要です。

## ローカルで開く

`index.html` をブラウザでダブルクリックするだけで動作します。

## データ更新

選手データはコード内 `DATA`（§5）の1か所に集約。負傷・追加招集・背番号確定があればここを書き換えるか、画面の「JSON読込」で差し替え可能。
